The Journey

Phase 1 — Infrastructure as Code

EKS cluster, VPC, IAM, and all supporting AWS resources defined in Terraform. Pipeline-driven from day one — no manual AWS console changes.

Phase 2 — GitOps Delivery

ArgoCD manages everything running on the cluster. App deployments are pull-based — ArgoCD syncs from Git, not pushed by a pipeline.

Phase 3 — AI-Assisted Platform Engineering

Claude Code used as a collaborative engineer throughout — not just for code generation, but for infrastructure decisions, debugging, and security review. A custom security auditor sub-agent runs on every push, triaging IAM, Kubernetes manifests, Dockerfiles, and pipeline configs. The goal: building a reusable AI harness that meaningfully accelerates how platform work gets done.

Phase 4 — Single Button Push

One pipeline run brings the entire platform from zero to live. One manual trigger tears it down at end of day. Cost to idle: near $0.

Building on the Platform

The platform is designed for self-service. Adding a new application requires no changes to the infrastructure layer — a developer creates a repo from the app-template, writes their app, and gets a live URL on the cluster automatically.

1 — Create repo from template

The app-template provides a hardened scaffold out of the box: GitHub Actions build pipeline, Dependabot with auto-merge (patch/minor only), pinned action SHAs, unprivileged nginx container, and Kubernetes manifests with resource limits and a full securityContext.

2 — Push app code

Any push to main triggers the build pipeline: image built, pushed to GHCR, and k8s/deployment.yaml updated with the exact commit SHA. No floating tags, ever.

3 — SRE adds one GitOps entry

A single ApplicationSet entry in the gitops repo tells ArgoCD to watch the new app repo. From that point on, ArgoCD owns the deploy — every commit to main syncs to the cluster automatically.

4 — App is live

External DNS writes a Route53 record. The AWS Load Balancer Controller provisions an ALB. The app is reachable at app-name.boudreauxlabs.com with HTTPS, no manual steps.
