About

Boudreaux Labs is a home lab built and maintained by Mr. Boudreaux, a Platform Engineering leader focused on cloud infrastructure, AI, developer tooling, and GitOps delivery patterns.

This site runs on the infrastructure it describes — deployed to Amazon EKS via ArgoCD, with every change driven through a GitHub Actions pipeline. No manual steps after the initial cluster apply.

Stack

AWS EKS Terraform ArgoCD GitHub Actions AWS Load Balancer Controller External DNS IRSA / OIDC Helm nginx Docker Route53 ACM

The Journey

Phase 1 — Infrastructure as Code

EKS cluster, VPC, IAM, and all supporting AWS resources defined in Terraform. Pipeline-driven from day one — no manual AWS console changes.

Phase 2 — GitOps Delivery

ArgoCD manages everything running on the cluster. App deployments are pull-based — ArgoCD syncs from Git, not pushed by a pipeline.

Phase 3 — AI-Assisted Platform Engineering

Claude Code used as a pair programmer throughout. Infrastructure decisions, pipeline authoring, and debugging — all assisted by AI tooling in the IDE.

Phase 4 — Single Button Push

One pipeline run brings the entire platform from zero to live. One manual trigger tears it down at end of day. Cost to idle: near $0.
